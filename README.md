# 🚀 Resume Ranking System (Microservices + AI Based)

A full-stack microservices-based Resume Ranking System that analyzes resumes using AI logic, ranks candidates, and provides improvement suggestions. The system also supports CSV report downloads.

---

## 🧠 Features

- 📄 Resume Upload (PDF support)
- 🤖 AI-based Resume Scoring
- 📊 Candidate Ranking System
- 💡 Resume Improvement Suggestions
- 📥 CSV Report Download
- 🔗 Microservices Architecture

---

## 🏗️ Architecture

- 🔍 Discovery Server (Eureka)
- 🚪 API Gateway
- 👤 User Service
- 📄 Resume Service
- 🏆 Ranking Service
- 🌐 Frontend (React)

---

## ⚙️ Tech Stack

### Backend:
- Java
- Spring Boot
- Spring Cloud (Eureka, Gateway)
- REST APIs

### Frontend:
- React.js
- Axios

### Database:
- MySQL
